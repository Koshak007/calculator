##Введение 
**Калькулятор индекса массы тела (ИМТ)** - это веб-приложение, предназначенное для быстрого и точного расчета индекса массы тела пользователя. Программа позволяет отслеживать динамику изменения веса, сохранять историю измерений и получать персонализированные рекомендации.

### Основные функции:
-  Расчёт имт по формуле: вес(кг)/рост(м)
-  Визуализация результатов на цветовой шкале
-  Сохранение истории изменений
-  Экспорт данных в CVS формате
-  Многопользовательский режим

---

## Примеры использования 
### 1. **Регистрация нового пользователя**
> При нажатии на кнопку "Зарегистрироваться" открывается форма для ввода имени (ника), возраста, пола и начальных параметров тела.

### 2. **Ввод текущих измерений**
> При нажатии на кнопку "Добавить измерение" появляются поля для ввода веса (кг) и роста (см).

### 3. **Расчёт ИМТ**
> При нажатии на кнопку "Рассчитать" система автоматически вычисляет ИМТ и отображает результат с точностью до десятых.

### 4. **Просмотр категорий веса**
> При нажатии на кнопку "Определить категорию" программа классифицирует результат согласно мировомы стандарту:
 > - дефицит массы тела — менее 18,5;
 > - нормальная масса тела — 18,5–24,9;
 > - избыточная масса тела (предожирение) — 25,0–29,9;
 > - ожирение первой степени — 30,0–34,9;

### 5. **Сохранение результата**
> При нажатии на кнопку "Сохранить" текущее измерение добавляется в историю с автоматической фиксацией даты и времени.

### 6. **Просмотр истории**
> При нажатии кнопки "История" открывается таблица со всеми сохраненными измерениями пользователя.

### 7. **Визуализация динамики**
> При нажатии на кнопку "График" система строит интерактивный график изменения веса за последние 30 дней.

### 8. **Экспорт данных**
> При нажатии кнопки "Экспорт в CSV" система формирует файл со всей историей изменений для скачивания.

### 9. **Смена пользователя**
> При нажатии кнопки "Смена пользователя" происходит выход из ьтекущего профиля и возврат к стартовому экрану выюора пользователя.

### 10. **Получение рекомендаций**
> При нажатии кнопки "Рекомендации" система генерирует персонализированные советы по питанеию и физической активности на основе текущего ИМТ. ( С премичанием что этот ответ сгенерировани автоматически и лучше проконсультироваться со спейциалистом)
